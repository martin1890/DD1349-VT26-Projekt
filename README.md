# DD1349-VT26-Projekt
procederually generate dungeon rooms

Using the Godot game engine, we will create a SEED based dungeon room generator that can be applied as a component to larger PCG projects. The rooms will be able to be instantiated with parameters to constrain the generated enviorment to include different dynamic and interactable elements, vary in length and shape, as well as verticality and complexity. The dungeon rooms will be generated to be guaranteed to be solveable (navigatable from start to end).

the planned dynamic elements include

pendulum schytes 
trapdoors
arrow dispensers
rolling boulders
climbable walls
rotating bridges
preasure plates/levers/buttons

We expect this project to be constrained to Godot, as it is intended as a development tool rather than a consumer product. By this we mean that you will need to install Godot to implement the dungeon room generator into a project.
